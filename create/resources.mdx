## Overview

Resources let you attach additional files to your units. These can include supplementary materials like worksheets, templates, or code samples.

## Adding Resources

1. When creating or editing a unit
2. Choose the Resources option
3. Upload a ZIP file
4. The resource will be available for download in that unit

## File Requirements

- Files must be packaged in a ZIP format
- Maximum file size: 4GB
- Can contain any type of file

## Best Practices

- Use clear file names
- Include a README if needed
- Organise files within your ZIP logically
- Mention the resource in your unit content

## Common Uses

- Code samples
- Exercise files
- Templates
- Documentation
- Extra reading materials
